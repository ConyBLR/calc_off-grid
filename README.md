# ☀ Calculadora Solar Off-Grid

Calculadora técnica para el dimensionamiento de sistemas fotovoltaicos aislados (off-grid).

Permite estimar la cantidad de paneles solares, banco de baterías, inversor, regulador de carga y protecciones eléctricas necesarias para una instalación solar fuera de red.

## Metodología

Los cálculos están basados en:

> Alonso, M. *"Gestión de las Energías Renovables: Perspectivas de Futuro"*
> 7ª edición · Módulo VI: Energía Solar Fotovoltaica · CIEMAT

## Estado

⚠ **Versión beta — en desarrollo.**
Los resultados son orientativos y no reemplazan el criterio de un profesional matriculado.
Ante cualquier duda o para un dimensionamiento definitivo, no dudes en consultar.
